Contains classes related to the representation and handling of BML specifications.

This package mainly provides the visitors for BML specification expressions ({@link b2bpl.bytecode.bml.BMLExpressionVisitor BMLExpressionVisitor}) and for BML store references appearing in modifies clauses ({@link b2bpl.bytecode.bml.BMLStoreRefVisitor BMLStoreRefVisitor}). In addition, the {@link b2bpl.bytecode.bml.SpecificationDesugarer SpecificationDesugarer} interface for desugaring BML specifications is defined along with a standard implementation of that interface. The actual classes representing the different kinds of BML specifications are contained in the sub-package {@link b2bpl.bytecode.bml.ast}.
